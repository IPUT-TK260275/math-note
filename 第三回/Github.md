## 1. 行列はなぜ重要か

行列は、たくさんの数をまとめて扱うための道具である。

例えば、次のようなものを行列で表せる。

- 連立方程式の係数
- 画像の画素の並び
- 画像処理に使う変換
- 平面や空間の回転、拡大、縮小

画像も数の集まりとして考えられるため、画像処理も行列の計算として扱える。

行列の積は、左の行列の「行」と右の行列の「列」を対応させて計算する。

```text
[a  b] [e  f]   [ae + bg   af + bh]
[c  d] [g  h] = [ce + dg   cf + dh]
```

### ポイント

左の「行」と右の「列」を対応させる。

## 3. 行列の積では順番が大事

普通の数では、かけ算の順番を変えても答えは同じである。

```text
2 × 3 = 3 × 2
```

しかし、行列では一般に順番を変えると答えが変わる。

```text
AB ≠ BA
```

### 例

```text
A = [ 4  0]    B = [1  0]
    [-3  1]        [0  2]
```

まず、`AB` を計算する。

```text
AB = [ 4  0] [1  0]
     [-3  1] [0  2]

   = [ 4  0]
     [-3  2]
```

次に、`BA` を計算する。

```text
BA = [1  0] [ 4  0]
     [0  2] [-3  1]

   = [ 4  0]
     [-6  2]
```

したがって、

```text
AB ≠ BA
```

である。

## 4. 逆行列の復習

数では、`a` に対して

```text
a × 1/a = 1
```

となる `1/a` を逆数という。

行列にも、逆数のような役割をする行列がある。それを **逆行列** という。

行列 `A` に対して、

```text
AX = XA = E
```

となる行列 `X` があるとき、`X` を `A` の逆行列といい、

```text
X = A^-1
```

と書く。

つまり、

```text
AA^-1 = E
A^-1A = E
```

である。

ここで、`E` は単位行列である。

  <tr><td>0</td><td>1</td></tr>
</table>

## 5. 2次正方行列の逆行列

2次正方行列

<table>
  <tr><td>a</td><td>b</td></tr>
  <tr><td>c</td><td>d</td></tr>
</table>

の行列式は、

```text
det A = ad - bc
```

である。

`det A ≠ 0` なら、逆行列は次の公式で求められる。

```text
A^-1 = 1 / det A × [ d  -b]
                    [-c   a]
```

### 例

```text
A = [1  2]
    [3  4]
```

行列式を求める。

```text
det A = 1 × 4 - 3 × 2
      = 4 - 6
      = -2
```

公式に代入する。

```text
A^-1 = 1 / (-2) × [ 4  -2]
                    [-3   1]

     = [ -2      1]
       [3/2  -1/2]
```

## 6. 逆行列と連立方程式

次の連立方程式を考える。

```text
x + y = 0
3x - 5y = 16
```

これは、行列を使って次のように書ける。

```text
[1   1] [x] = [ 0]
[3  -5] [y]   [16]
```

ここで、

```text
A = [1   1]    B = [x]    C = [ 0]
    [3  -5]        [y]        [16]
```

とおくと、

```text
AB = C
```

と書ける。

両辺に左から `A^-1` をかける。

```text
A^-1AB = A^-1C
```

`A^-1A = E` なので、

```text
EB = A^-1C
```

したがって、

```text
B = A^-1C
```

となる。

### ポイント

連立方程式を行列で解くときは、次の形を目指す。

```text
[x] = A^-1 [右辺1]
[y]        [右辺2]
```

---

## 7. 例題：連立方程式を逆行列で解く

次の連立方程式を解く。

```text
x + y = 0
3x - 5y = 16
```

行列で表す。

```text
[1   1] [x] = [ 0]
[3  -5] [y]   [16]
```

係数行列を

```text
A = [1   1]
    [3  -5]
```

とする。

### 1. 逆行列を求める

```text
det A = 1 × (-5) - 1 × 3
      = -5 - 3
      = -8
```

したがって、

```text
A^-1 = 1 / (-8) × [-5  -1]
                     [-3   1]

     = 1 / 8 × [5   1]
               [3  -1]
```

### 2. B = A^-1C を計算する

```text
B = A^-1C

  = 1 / 8 × [5   1] [ 0]
            [3  -1] [16]

  = 1 / 8 × [ 16]
            [-16]

  = [ 2]
    [-2]
```

したがって、

```text
x = 2
y = -2
```

である。

## 8. 解が無限にある場合

次の連立方程式を考える。

```text
x - 4y = 2
2x - 8y = 4
```

2つ目の式は、1つ目の式を2倍しただけである。

```text
2 × (x - 4y = 2)
→ 2x - 8y = 4
```

つまり、実質的には式が1本しかない。

```text
x - 4y = 2
```

から、

```text
x = 2 + 4y
```

となる。

`y` には好きな値を代入できるので、解は無限にある。

```text
y = 0, 1, 2, 3, ...

[x] = [ 2], [ 6], [10], ...
[y]   [ 0]  [ 1]  [ 2]
```

### 行列で見ると

```text
[1  -4] [x] = [2]
[2  -8] [y]   [4]
```

係数行列を

```text
A = [1  -4]
    [2  -8]
```

とすると、

```text
det A = 1 × (-8) - (-4) × 2
      = -8 - (-8)
      = 0
```

したがって、`A^-1` は存在しない。

逆行列の公式を使おうとすると、

```text
1 / det A = 1 / 0
```

が出てくる。これは計算できない。

## 9. 練習問題

### 問題1

```text
x - 2y = 0
2x + y = 15
```

行列で表す。

```text
[1  -2] [x] = [ 0]
[2   1] [y]   [15]
```

係数行列を

```text
A = [1  -2]
    [2   1]
```

とする。

```text
det A = 1 × 1 - (-2) × 2
      = 1 + 4
      = 5
```

したがって、

```text
A^-1 = 1 / 5 × [ 1  2]
                  [-2  1]
```

よって、

```text
[x] = 1 / 5 × [ 1  2] [ 0]
[y]           [-2  1] [15]

    = 1 / 5 × [30]
              [15]

    = [6]
      [3]
```

答えは、

```text
x = 6
y = 3
```

である。

### 問題2

```text
2x + 3y = 8
3x - 2y = -1
```

行列で表す。

```text
[2   3] [x] = [ 8]
[3  -2] [y]   [-1]
```

係数行列を

```text
A = [2   3]
    [3  -2]
```

とする。

```text
det A = 2 × (-2) - 3 × 3
      = -4 - 9
      = -13
```

したがって、

```text
A^-1 = 1 / (-13) × [-2  -3]
                     [-3   2]

     = 1 / 13 × [2   3]
                [3  -2]
```

よって、

```text
[x] = 1 / 13 × [2   3] [ 8]
[y]            [3  -2] [-1]

    = 1 / 13 × [16 - 3]
               [24 + 2]

    = 1 / 13 × [13]
               [26]

    = [1]
      [2]
```

答えは、

```text
x = 1
y = 2
```

である。

---

## 10. 今回のまとめ

- 行列の積は、左の「行」と右の「列」で計算する
- 行列の積は、一般に `AB ≠ BA` である
- 逆行列は、かけると単位行列になる行列である
- 2次正方行列の逆行列は、行列式が `0` でないときだけ求められる
- 連立方程式を `AB = C` の形にできれば、`B = A^-1C` で解ける

---

## 用語

| 用語 | 意味 |
| --- | --- |
| 行列の積 | 左の行と右の列を対応させて計算する積 |
| `AB ≠ BA` | 行列では積の順番を変えると答えが変わることがある |
| 逆行列 | かけると単位行列になる行列 |
| 単位行列 | 数の `1` のような役割をする行列 |
| `det A` | 行列 `A` の行列式 |
| `det A ≠ 0` | 逆行列が存在する条件 |
| `det A = 0` | 逆行列が存在しない条件 |
