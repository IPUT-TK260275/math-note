# 線形代数 1：行列の基礎ノート

## 1. 授業の目的

この授業では、線形代数の入口として **行列** を学ぶ。

行列は、AI、画像処理、データ処理、プログラミング、専門科目などで使われる大事な考え方である。

今日の目標は、次のことができるようになること。

* 行列とは何かを理解する
* 行列の大きさを読めるようにする
* 成分の場所を指定できるようにする
* 行列の足し算・引き算ができるようにする
* 行列の掛け算のルールを理解する
* 行列では、掛け算の順番が大事だと理解する

---

## 2. 行列とは

行列とは、数字を **行** と **列** に並べたもの。

例：

```text
[1  2]
[3  4]
```

このように、数字をカッコで囲んで書く。

---

## 3. 行と列

### 行

横方向を **行** という。

```text
[1  2]
[3  4]
```

この行列では、

* `[1  2]` が **1行目**
* `[3  4]` が **2行目**

### 列

縦方向を **列** という。

```text
[1  2]
[3  4]
```

この行列では、

* `1, 3` が **1列目**
* `2, 4` が **2列目**

### 覚え方

行列の大きさは、必ず

```text
行 × 列
```

の順番で読む。

つまり、**先に行、あとに列**。

---

## 4. 行列の大きさ

### 例1

```text
[1  2]
[3  4]
```

これは、

* 2行
* 2列

なので、

```text
2 × 2 行列
```

という。

また、正方形の形なので **2次正方行列** ともいう。

---

### 例2

```text
[1  2  3]
[4  5  6]
[7  8  9]
```

これは、

* 3行
* 3列

なので、

```text
3 × 3 行列
```

または **3次正方行列** という。

---

### 例3

```text
[1  2  3]
[4  5  6]
```

これは、

* 2行
* 3列

なので、

```text
2 × 3 行列
```

という。

正方形ではないので、**長方行列** と呼ぶこともある。

---

## 5. 成分とは

行列の中に入っている一つ一つの数字を **成分** という。

例：

```text
A =
[1  2]
[3  4]
[5  6]
```

このとき、

* `1`, `2`, `3`, `4`, `5`, `6` が成分

---

## 6. 成分の場所の読み方

成分の場所は、

```text
行番号, 列番号
```

で表す。

### 例

```text
A =
[1  2]
[3  4]
[5  6]
```

### 問題

`A` の `(3,1)` 成分の値は？

### 解き方

`(3,1)` 成分とは、

* 3行目
* 1列目

の数字を指す。

```text
A =
[1  2]
[3  4]
[5  6]  ← 3行1列目は 5
```

したがって、

```text
(3,1) 成分 = 5
```

---

## 7. 行列の足し算

行列の足し算は、**同じ場所の成分どうしを足す**。

### 例

```text
A =
[1  2]
[3  4]

B =
[5  6]
[7  8]
```

```text
A + B
=
[1  2] + [5  6]
[3  4]   [7  8]
```

同じ場所どうしを足す。

```text
=
[1+5  2+6]
[3+7  4+8]
```

```text
=
[ 6   8]
[10  12]
```

### ポイント

足し算できるのは、行列の大きさが同じときだけ。

```text
2 × 2 + 2 × 2
```

は計算できる。

しかし、

```text
2 × 2 + 3 × 2
```

のように形が違うものは計算できない。

---

## 8. 行列の引き算

行列の引き算も、**同じ場所の成分どうしを引く**。

### 例

```text
A =
[1  2]
[3  4]

B =
[5  6]
[7  8]
```

```text
B - A
=
[5  6] - [1  2]
[7  8]   [3  4]
```

```text
=
[5-1  6-2]
[7-3  8-4]
```

```text
=
[4  4]
[4  4]
```

---

## 9. 行列の前に数字があるとき

行列の前に数字があるときは、行列の中のすべての成分にその数字をかける。

### 例

```text
4 [1  1]
  [1  1]
=
[4  4]
[4  4]
```

逆に、すべての成分に共通の数字があるときは、外に出せる。

```text
[4  4]
[4  4]
=
4 [1  1]
  [1  1]
```

---

## 10. 例題：2B - 3A

```text
A =
[1  2]
[3  4]

B =
[5  6]
[7  8]
```

```text
2B - 3A
=
2 [5  6] - 3 [1  2]
  [7  8]     [3  4]
```

まず、それぞれ中にかける。

```text
=
[10  12] - [3   6]
[14  16]   [9  12]
```

同じ場所どうしを引く。

```text
=
[10-3  12-6]
[14-9  16-12]
```

```text
=
[7  6]
[5  4]
```

---

## 11. 単位行列

対角成分がすべて `1`、それ以外が `0` の行列を **単位行列** という。

### 2次の単位行列

```text
E =
[1  0]
[0  1]
```

### 3次の単位行列

```text
E =
[1  0  0]
[0  1  0]
[0  0  1]
```

### ポイント

単位行列は、普通の数でいう **1** のような役割をする行列。

---

## 12. ゼロ行列

すべての成分が `0` の行列を **ゼロ行列** という。

例：

```text
O =
[0  0]
[0  0]
```

### 例

同じ行列を引くと、ゼロ行列になる。

```text
A - A = O
```

具体的には、

```text
[1  2] - [1  2]
[3  4]   [3  4]
=
[0  0]
[0  0]
```

---

## 13. 行列の掛け算ができる条件

行列の掛け算は、足し算・引き算よりルールが厳しい。

```text
(m × n) × (n × p)
```

のように、**左の行列の列数** と **右の行列の行数** が同じときだけ計算できる。

### 例

```text
(3 × 2) × (2 × 4)
```

真ん中の `2` と `2` が同じなので計算できる。

答えの形は、外側の数字を使って、

```text
3 × 4
```

になる。

### 形の見方

```text
(3 × 2) × (2 × 4)
     ↑      ↑
  ここが同じなら計算できる
```

真ん中が一致するので計算できる。

残った外側が答えの形。

```text
3 × 4
```

---

## 14. 掛け算できない例

```text
(3 × 2) × (1 × 4)
```

この場合、真ん中が

```text
2 ≠ 1
```

なので計算できない。

---

## 15. 行列の掛け算のやり方

行列の掛け算は、

* 左の行列は **横** に見る
* 右の行列は **縦** に見る

これが基本。

### 例

```text
[1  2] [5  6]
[3  4] [7  8]
```

答えは `2 × 2` 行列になる。

```text
[?  ?]
[?  ?]
```

---

### 左上の成分

左上は、

* 左の行列の1行目
* 右の行列の1列目

を使う。

```text
1 × 5 + 2 × 7
= 5 + 14
= 19
```

---

### 右上の成分

右上は、

* 左の行列の1行目
* 右の行列の2列目

を使う。

```text
1 × 6 + 2 × 8
= 6 + 16
= 22
```

---

### 左下の成分

左下は、

* 左の行列の2行目
* 右の行列の1列目

を使う。

```text
3 × 5 + 4 × 7
= 15 + 28
= 43
```

---

### 右下の成分

右下は、

* 左の行列の2行目
* 右の行列の2列目

を使う。

```text
3 × 6 + 4 × 8
= 18 + 32
= 50
```

---

### したがって

```text
[1  2] [5  6]   [19  22]
[3  4] [7  8] = [43  50]
```

---

## 16. 掛け算の覚え方

行列の掛け算は、

```text
左は横、右は縦
```

で見る。

そして、対応する数字をかけて、最後に足す。

### 左上なら

```text
左の1行目 × 右の1列目
```

### 右上なら

```text
左の1行目 × 右の2列目
```

### 左下なら

```text
左の2行目 × 右の1列目
```

### 右下なら

```text
左の2行目 × 右の2列目
```

---

## 17. 行列の掛け算は順番が大事

普通の数字では、

```text
2 × 3 = 3 × 2
```

となる。

しかし、行列では基本的に

```text
AB ≠ BA
```

になる。

つまり、**Aを先にかけるか、Bを先にかけるかで答えが変わる**。

---

## 18. 例：AB と BA の違い

```text
A =
[ 4  0]
[-3  1]

B =
[1  0]
[0  2]
```

---

### AB

```text
AB
=
[ 4  0] [1  0]
[-3  1] [0  2]
```

```text
=
[ 4  0]
[-3  2]
```

---

### BA

```text
BA
=
[1  0] [ 4  0]
[0  2] [-3  1]
```

```text
=
[ 4  0]
[-6  2]
```

---

### 比較

```text
AB =
[ 4  0]
[-3  2]

BA =
[ 4  0]
[-6  2]
```

左下の成分が違う。

したがって、

```text
AB ≠ BA
```

---

## 19. (A+B)(A-B) の展開

普通の数なら、

```text
(A+B)(A-B) = A^2 - B^2
```

となる。

しかし、行列では注意が必要。

行列では、掛け算の順番が変わると答えも変わるため、

```text
AB ≠ BA
```

だからである。

---

### 展開する

```text
(A+B)(A-B)
```

左側の `A+B` を、それぞれ右側にかける。

```text
= A(A-B) + B(A-B)
```

さらに展開する。

```text
= A^2 - AB + BA - B^2
```

---

### 重要ポイント

ここで、

```text
-AB + BA
```

は、基本的に `0` にならない。

なぜなら、

```text
AB ≠ BA
```

だから。

したがって、行列では

```text
(A+B)(A-B) = A^2 - AB + BA - B^2
```

であり、普通の数のように簡単に

```text
A^2 - B^2
```

とはできない。

---

## 20. 例：(A+B)(A-B) の計算

```text
A =
[ 4  0]
[-3  1]

B =
[1  0]
[0  2]
```

このとき、

```text
(A+B)(A-B)
= A^2 - AB + BA - B^2
```

各部分は、

```text
A^2 =
[ 16  0]
[-15  1]
```

```text
AB =
[ 4  0]
[-3  2]
```

```text
BA =
[ 4  0]
[-6  2]
```

```text
B^2 =
[1  0]
[0  4]
```

したがって、

```text
A^2 - AB + BA - B^2
```

```text
=
[ 16  0] - [ 4  0] + [ 4  0] - [1  0]
[-15  1]   [-3  2]   [-6  2]   [0  4]
```

```text
=
[ 15   0]
[-18  -3]
```

## 21. ノートの取り方のポイント

先生が強調していた大事なこと。

### 1. 計算過程を必ず書く

数学が苦手な人ほど、途中式を飛ばして間違えやすい。

上のクラスの学生は、計算過程を丁寧に書いている。

だから、途中式を省略しないことが大事。

---

### 2. イコールを縦にそろえる

横にどんどん書いていくと、どこで何を計算したのか分かりにくくなる。

例：

```text
A + B
=
[1  2] + [5  6]
[3  4]   [7  8]
```

```text
=
[1+5  2+6]
[3+7  4+8]
```

```text
=
[ 6   8]
[10  12]
```

このように、イコールを縦にそろえると見やすい。

---

### 3. 行列は大きく書く

行列は数字が近すぎると、どの数字がどの場所にあるのか分からなくなる。

そのため、行列は少し大きめに、余白を空けて書く。
