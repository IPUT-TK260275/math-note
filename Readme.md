## ファイルの使い分け

各回のフォルダには、主に次のファイルを置く。

| ファイル | 用途 |
| --- | --- |
| `NoteN.md` | LaTeX 数式を使った通常版ノート |
| `Github.md` | GitHub で崩れにくい表示用ノート |

## GitHub表示用ノートについて

GitHub では、環境によって LaTeX の行列表示が崩れることがある。

そのため、`Github.md` では行列を HTML テーブルやコードブロックで表示している。GitHub 上で読む場合は、各回の `Github.md` を使う。
