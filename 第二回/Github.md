## 1. 行列式とは

2次正方行列

<table>
  <tr><td>a</td><td>b</td></tr>
  <tr><td>c</td><td>d</td></tr>
</table>

の行列式は、次のように求める。

```text
det A = ad - bc
```

覚え方：

```text
左上 × 右下 - 右上 × 左下
```

---

## 2. 行列式の例

### 例1

```text
|8| = 8
```

1つだけの数からなる行列の行列式は、その数そのものである。

### 例2

```text
|-2| = -2
```

これも 1次の行列式なので、その数そのものになる。

### 例3

<table>
  <tr><td>5</td><td>-6</td></tr>
  <tr><td>7</td><td>-8</td></tr>
</table>

計算：

```text
5×(-8) - (-6)×7
= -40 - (-42)
= 2
```

## 3. 練習：行列式を求める

### 問題1

## 4. 行列の掛け算は順番が大事

行列の掛け算では、普通の数と違って、順番を変えると結果が変わることがある。

```text
AB ≠ BA
```

例：

```text
A = [ 4  0]    B = [1  0]
    [-3  1]        [0  2]
```

```text
AB = [ 4  0]
     [-3  2]

BA = [ 4  0]
     [-6  2]
```

したがって、

```text
AB ≠ BA
```

である。

---

## 5. 逆行列とは

行列にも、数でいう「逆数」のようなものがある。
それを **逆行列** という。

行列 `A` に対して、

```text
AA^-1 = E
A^-1A = E
```

となる行列 `A^-1` を、**A の逆行列** という。

ここで `E` は単位行列である。

<table>
  <tr><td>1</td><td>0</td></tr>
  <tr><td>0</td><td>1</td></tr>
</table>

単位行列は、数でいう 1 のような役割をする。

---

## 6. 2次正方行列の逆行列の公式

行列

<table>
  <tr><td>a</td><td>b</td></tr>
  <tr><td>c</td><td>d</td></tr>
</table>

に対して、逆行列は次の公式で求められる。

```text
A^-1 = 1 / det A × [ d  -b]
                      [-c   a]
```

ただし、

```text
det A ≠ 0
```

のときだけ逆行列が存在する。

---

## 7. 逆行列を求める例題

```text
A = [1  2]
    [3  4]
```

まず行列式を求める。

```text
det A
= 1×4 - 2×3
= 4 - 6
= -2
```

次に公式に当てはめる。

```text
A^-1
= 1 / (-2) × [ 4  -2]
              [-3   1]
= [ -2      1]
  [3/2  -1/2]
```

---

## 8. 逆行列の確かめ

本当に逆行列になっているかは、掛け算して単位行列になるかで確かめる。

```text
AA^-1 = E
A^-1A = E
```

の両方が成り立てばよい。

---

## 9. 逆行列が存在しない場合

```text
C = [ 1  -1]
    [-3   3]
```

行列式を求める。

```text
det C
= 1×3 - (-1)×(-3)
= 3 - 3
= 0
```

今回は `det C = 0` なので、逆行列の公式に `1 / 0` が出てきてしまう。
これは計算できない。

したがって、

```text
C^-1 は存在しない
```

---

## 10. 逆行列が存在する条件

2次正方行列に対して、逆行列が存在する条件は、

```text
det A ≠ 0
```

である。

反対に、

```text
det A = 0
```

なら逆行列は存在しない。

---

## 今日の最重要まとめ

- 2次正方行列の行列式は `det A = ad - bc`
- 逆行列は、掛けると単位行列になる行列
- 2次正方行列の逆行列は公式で求められる
- 逆行列を求める前に、必ず行列式を確認する
- `det A = 0` のとき逆行列は存在しない
